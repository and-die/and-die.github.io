Freie Software, einfach erklärt.

Was ist eigentlich „Freie Software“

Freie Software bedeutet nicht, dass die Software kostenlos ist, so wie viele Leute denken. Freie Software kann nämlich auch etwas kosten. Freie Software ist Software, bei der der Nutzer die volle Macht über ein Programm hat und dieses nach seinem Belieben verändern und weiter entwickeln kann. Ebenfalls muss der Nutzer erfahren, was das Programm macht, z.B. muss er den Quellcode¹ einsehen können, um zu wissen, dass das Programm keine Daten klaut. Freie Software muss vier Freiheiten/Bedingungen erfüllen, damit es freie Software genannt werden kann, diese findet ihr unten.

Die vier Freiheiten

Freie Software muss „Die vier Freiheiten“ erfüllen, sonst ist es keine freie Software. Hier die vier Freiheiten (in meinen Worten):

  • Freiheit 1: Man muss das Programm verwenden können wie man möchte.
  • Freiheit 2: Man muss das Programm verstehen können.
  • Freiheit 3: Man darf das Programm weiterverbreiten, um damit anderen Menschen zu helfen.
  • Freiheit 4: Man darf das Programm verbessern und diese Verbesserungen der Öffentlichkeit freigeben, wenn man es unter der selben Lizenz² veröffentlicht.

Proprietäre Software

Wenn mindestens eine von den vier Freiheiten nicht erfüllt ist, nennt man diese Software dann proprietäre Software. Bei dieser weiß man z.B. nicht was sie macht, da der Quellcode oft nicht frei zugänglich ist. Der Entwickler kann also z.B. Schadsoftware auf das Gerät des Nutzers installieren, welche z.B. Daten vom Nutzer klaut. Viele große Unternehmen schreiben ihre Nutzungsbedingungen so geschickt, dass der Nutzer sie nicht verstehen kann und meistens direkt auf "Weiter" klickt. Auf diese Weise können sie in ihre Nutzungsbedingungen hineinschreiben, dass sämtliche Daten des Nutzers und anderer Personen (die in Kontakt mit ihm stehen) an das Unternehmen gesendet werden. Diese Daten werden dann meistens verkauft.

Open Source

Wenn von den vier Freiheiten die letzte nicht erfüllt ist, sonst aber alle anderen nennt man man dies Open Source. Bei Open Source darf man das Programm verwenden, wie man möchte, kann das Programm verstehen und darf das Programm weiterverbreiten, um damit anderen Leuten zu helfen. Jedoch darf man das Programm nicht verändern und dann wieder veröffentlichen.

Freie- und Unfreie-Dienste

Es gibt freie und unfreie Dienste, z.B der Messengerdienst Watsapp ist ein unfreier Dienst, den wenn man die Nutzungsbedingung gelesen hat, weiß man, das Whatsapp z.B. aufs Telefonbuch des Handys zugreifen kann. Die meisten Leute lesen die Nutzungsbedingung gar nicht, sondern klicken direkt auf den Button zum weitermachen. Man kann aber z.B. als Alternative freie Dienste wie Movim benutzen.

Freie Hardware³

Freie Hardware ist Hardware, die nach frei zugänglichen Bauplänen hergestellt wird. Die Bewegung und die Idee von freier Hardware steht der Idee der freien Software nahe, bzw. geht auf diese zurück.

Jugendverein für freie Software

Der Teckids e.V. ist ein Jugendverein, der sich für freie Software (vor allem in Schulen) einsezt. Er organisiert Jugendfreizeiten, die z.B. auf Messen fahren und sich diese anschauen. Beim Teckids e.V. wird fast alles von Kindern gemacht.

Hier ein Link zu der Webseite von Teckids: https://www.teckids.org/de/

¹)Der Quellcode eines Programms ist das, was der Programmierer oder die Programmiererin in einer Programmiersprache schreibt. Dieser ist dann noch für Menschen lesbar. Im nächsten Schritt wird der Quellcode dann in einen Maschinencode umgewandelt, dieser ist dann nicht mehr von Menschen lesbar.

²)Eine Lizenz ist eine Art Regel. Eine Lizenz bestimmt, was der Nutzer mit dem Programm machen darf, z.B. ob er es verändern/verbessern darf oder ob er es veröffentlichen darf.

³)Hardware ist z.B. was sich in einem Computer befindet. Also die Technik in dem Computer.
